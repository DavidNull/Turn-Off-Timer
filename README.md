# ⌚ Turn-Off Timer  

**Turn-Off Timer** is a simple countdown tool for Windows 10 that shuts down your system automatically.  

## 🚀 Features  
- Two shutdown options:  
  1️⃣ **1 hour**  
  2️⃣ **2 hours**  
- Includes a ready-to-use **.exe** file (built with PyInstaller).  

## 📦 How to Use  
1. Download the `.exe` file.  
2. Run it. Done!
